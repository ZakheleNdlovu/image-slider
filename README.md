## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ZakheleNdlovu/image-slider.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd image-slider
   ```

## Usage

1. Open `index.html` in your browser to see the image slider in action.
2. You can replace the sample images in the `images` folder with your own.

## Customization

- **Add or Remove Slides:**  
  Edit the `<div class="slide">` elements in `index.html` to add or remove images.
- **Styling:**  
  Modify `style.css` to change the appearance, transition effects, or layout.
- **Image Sources:**  
  Place your images in the `images` folder and update the `<img>` tags accordingly.
